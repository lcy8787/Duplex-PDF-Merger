# PDF 雙面掃描合併工具

一款快速的客戶端工具，專為將單面掃描的奇數頁和偶數頁 PDF 合併並正確排序為完整文件而設計。完全支援 ADF（自動進紙器）掃描儀常見的倒序偶數頁。

---

## 遇到的問題

使用僅支援單面掃描的打印機或 ADF 掃描儀時，雙面文件需要分兩步進行：

1. **第一次**：以正常方向放入紙張，掃描所有奇數頁（1, 3, 5...）。

2. **第二次**：翻轉紙疊，從背面以倒序（6, 4, 2...）掃描剩餘的偶數頁。

結果會產生兩個獨立的 PDF 文件，頁面順序混亂——這正是此工具要解決的問題。

---

## 功能特點

- **純客戶端運行** — 無需上傳，無需伺服器，您的文件不會離開您的設備
- **自動頁面排序** — 無縫交錯合併奇數頁和偶數頁
- **倒序偶數頁支援** — 內建 ADF 反向掃描選項
- **離線運行** — 開啟 HTML 文件後，無需網絡連接
- **無需安裝** — 在任何現代瀏覽器中直接開啟 HTML 文件即可

---

## 使用方法

1. 在瀏覽器中開啟 `printer adf duplex scan merger.html`。

2. 選擇包含**奇數頁**（1, 3, 5...）的 PDF 檔案。

3. 選擇包含**偶數頁**（如果是倒序則為 6, 4, 2...）的 PDF 檔案。

4. 如果偶數頁是以倒序掃描的（標準 ADF 工作流程），請保持勾選**「倒序偶數頁」**。

5. 輸入導出的檔案名稱（可選）。

6. 點擊**「合併並下載」**即可獲得完整文件。

---

## 技術細節

- 使用 [pdf-lib](https://pdf-lib.js.org/) 處理 PDF
- 單一 HTML 文件，無外部依賴（pdf-lib 從 CDN 加載；初次加載後可離線運行）
- 所有處理均在瀏覽器中完成
